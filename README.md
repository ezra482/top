# GlobalAI Knowledge Base
一个致力于整合全球知识、对标世界顶级的AI知识库项目，支持多语言、多领域知识检索与智能问答。

## 项目目标
- 构建覆盖**科学、文化、历史、宗教、技术**等全领域的知识图谱
- 实现多语言（联合国6种工作语言+主流语种）智能问答
- 打造开源、可扩展的AI框架，支持全球开发者协作贡献

## 技术栈
- 前端：Next.js + React + Tailwind CSS（支持SSR/SSG，全球访问加速）
- 后端：Node.js + Express（或Serverless函数）
- AI核心：开源大模型集成（如Llama、BERT）+ 自定义知识嵌入
- 数据存储：PostgreSQL + Redis（全球多区域部署）
- 部署：Vercel + GitHub Actions（自动CI/CD，全球CDN加速）

## 快速开始
```bash
# 克隆项目
git clone https://github.com/ezra482/global-ai-knowledge-base.git
cd global-ai-knowledge-base

# 安装依赖
npm install

# 本地启动
npm run dev
